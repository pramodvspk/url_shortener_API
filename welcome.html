URL Shortener API

User Stories

  1. The User can pass a URL as a parameter and will receive a shortened url in a JSON format.
  2. When the users visit the shortened url they are redirected to the original url

Example creation usage

https://pramodvspk-url-shortener.herokuapp.com/shorten/https://google.com

https://pramodvspk-url-shortener.herokuapp.com/shorten/https//facebook.com

Example creation output

{"original_url": "https://google.com", "short_url: "https://pramodvspk-url-shortener.herokuapp.com/1"}

Usage

https://pramodvspk-url-shortener.herokuapp.com/1

Will redirect to

https://google.com
